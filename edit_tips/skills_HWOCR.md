# 软件技巧：汉王PDF OCR 软件


## 安装

1. 汉王PDF OCR为免费软件

2. [下载地址](http://xiazai.zol.com.cn/detail/34/335364.shtml)


## 导入PDF（或图片）

1. 文件 => 打开图像 => 选择pdf文件

2. 分辨率（DPI）调整为600

	默认的分辨率太低，降低了识别准确率，调整为600会显著增加文字识别的精度。


## 识别

以下4个步骤依次循环。

1. 鼠标选择识别区域；
2. 识别
3. copy 至文本巨匠，修改错误
4. 取消选择


如果你有更好的方案，期待反馈。
