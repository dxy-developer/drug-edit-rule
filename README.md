# Editor Rules

形式应该与内容分离, 绝不允许在数据中标记样式. 



## 推荐的编辑风格


1. 使用英文符号撰写markdown标签
1. 不使用中文全角符号
1. 不使用\<br>, 使用markdown的｢段落｣替代
1. 使用有序列表
1. 每一行文字要顶齐, 不要预留空格
1. 每一段文字要顶齐, 不要预留空格


## 常用的语法

1. 列表
1. 嵌套列表
1. 加粗 **hello**
1. 段落
